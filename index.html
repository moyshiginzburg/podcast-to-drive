🎙 מדריך פשוט להתקנה

מה זה בכלל? זה כלי שמחבר בין פודקאסטים ל־Google Drive דרך Google Sheets, כדי שתוכלו לשמור פרקים בצורה מסודרת בדרייב.

הוא מוסיף לגיליון שלכם תפריט בשם 🎙 הסכתים, ומשם אפשר לפתוח מנהל פשוט לחיפוש פודקאסטים ולהורדת פרקים.

הדף הזה מיועד לאנשים לא טכנולוגיים: צעד־אחר־צעד, בעברית פשוטה, בלי בלבול.

המטרה: להתקין את מנהל ההסכתים בגיליון Google Sheets ולפתוח אותו מתוך התפריט.

שלב 1

📄 פתחו גיליון חדש

היכנסו ל־Google Drive, צרו Google Sheets חדש.

🖼️ צילום מסך: Google Drive → חדש (+) → Google Sheets
שלב 2

🧩 פתחו את עורך הסקריפט

בתוך הגיליון: Extensions (תוספים) → Apps Script.

🖼️ צילום מסך: התפריט העליון עם Extensions → Apps Script
שלב 3

📋 הדביקו את Code.gs

מחקו את הקוד שמופיע כברירת מחדל, והדביקו את תוכן הקובץ Code.gs מהריפו.

פתח Code.gs
🖼️ צילום מסך: עורך Apps Script אחרי הדבקה של Code.gs
שלב 4

🖼️ צרו קובץ HTML בשם Sidebar

לחצו על + ליד קבצים → HTML → תנו שם: Sidebar (בלי ‎.html).

הדביקו שם את תוכן Sidebar.html מהריפו.

פתח Sidebar.html
🖼️ צילום מסך: חלון יצירת HTML עם השם Sidebar
שלב 5

✅ שמירה והרשאות

לחצו Save, בחרו פונקציה onOpen, ואז Run. אשרו הרשאות כשתתבקשו.

🖼️ צילום מסך: בחירת onOpen ולחיצה על Run
שלב 6

🎉 סיימנו

חזרו לגיליון ורעננו את הדף. יופיע תפריט 🎙 הסכתים → פתח מנהל הסכתים.

אם התפריט לא מופיע מיד, רענון נוסף בדרך כלל פותר את זה.
🖼️ צילום מסך: תפריט 🎙 הסכתים בגיליון
